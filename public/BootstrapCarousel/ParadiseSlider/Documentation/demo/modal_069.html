mega vs single

You can use Paradise Slider CSS in two ways:

  • You can use only one Compressed or UnCompressed CSS file for all Paradise Slider layouts. You can find these files in the sub folder min in main css folder.
  • As well as concern to other method, you can use a single CSS file for a single Paradise Slider layout. You can find all of these files in the main CSS folder. In this case you can use only modal_069.css file.

Trigger Part

The modal plug-in toggles your hidden content on demand. A modal always appears when we click on some content. In this example we have used an Image Gallery to Trigger the Modal. You can also use link, button etc to trigger the Modal.

how it works?

To trigger the modal window, you need to use a button, link, image, heading etc.

Then include the two data-* attributes:

  • data-toggle="modal" opens the modal window
  • data-target="#myModal" points to the id of the modal. In above example as you can see we have used data-target="#ps_modal_069". In this layout we have assigned #ps_modal_069 Id to the Modal.

The below examples are showing that how can we trigger modal by using the different methods besides Image Gallery


<a href="#" data-toggle="modal" data-target="#ps_modal_069">Open Modal</a> <!-- By Using Link -->
		
<img src="image.jpg" alt="" data-toggle="modal" data-target="#ps_modal_069"> <!-- By Using Image -->
		
<button type="button" data-toggle="modal" data-target="#ps_modal_069">Open Modal</button> <!-- By Using Button -->

03 - Classes

Modal

We have used following our own classes and id in this layout, that control and change the whole Modal.

#ps_modal_069 : Describe the Modal's Unique Id.

.modal: Describe the content as a modal and brings focus to it.

.fade : Enable Transition Effect which Fades the modal In and Out. Remove this class if you do not want this effect.

.ps_modal_fadeInRight : Enable Fade In Right Effect for Modal. Without .fade class this Effect will not work.

.ps_modal_close_transp : This class makes Transparent Color Close Button at the Top and Left side of the Modal.

.modal-dialog : This sets the proper width and margin of the modal.

.modal-content : The div which has .modal-content as a class name contains the Slider

Slider

We have used following our own classes and id in this layout, that control and change the whole slider.

.swipe_x: This class enables Horizontal swipe function

#video_slider_modal_069 : Describe the Slider's Unique Id.

.vid_slider_modal_069_slide : Enable Styles for Text Layer

.vid_slider_modal_069_slide_right : Move the Text Layer to the Right side

.vid_slider_modal_069_slide_center : Move the Text Layer in the Center

.thumb_scroll_x: This class enables Horizontal scroll function for indicators

.vid_slider_modal_thumb : This class makes images thumbnails Horizontally to the bottom of the slider

.slide : Enable Sliding Effect.

.ps_easeInOutQuad : This class enables Transition Timing Function for Slide Effect

Video

We have used following our own classes and id in this layout, that control and change the video.

.vid_slider_modal_069: This class enables Style for Videos

control: Enables the video controls such as a play/pause button etc.

Note: vid Stands for " Video ", thumb Stands for " Thumbnail " and 069 means " Version 069 ". And ps stands for " Paradise Slider ".

In this layout we have used images thumbnails to move from one slide to another. We have created these thumbnails Horizontally and in the bottom of the slider. We have created the following additional functions to control the slides:

  1. .swipe_x

    This class enables swipe function. You can controll the slides by dragging the mouse button Horizontally and in touch devices this class also enables the touch control.
    If you want to know about other functions click HERE.

  2. .thumb_scroll_x

    This class enables Horizontal scroll function for indicators.
    If you want to know about other functions click HERE.

  3. .ps_easeInOutQuad

    This class enables Transition Timing Function for Slide Effect.
    If you want to know about other Transition Timing Function for Slide Effect please click HERE.

  4. .ps_modal_fadeInRight

    This class enables Fade In Right Effect for Modal.
    If you want to know about other Modal Effects please click HERE.

  5. .vid_slider_modal_thumb

    This class makes Horizontal images thumbnails in the bottom of the slider.
    If you want to know about other types of controls buttons please click HERE.

  6. data-pause="hover"

    This attribute pauses the slider from sliding through the next slide when the mouse pointer enters the slider, and when the mouse pointer leaves the slider it resumes the sliding.

  7. data-interval="false"

    This attribute specifies the delay (in milliseconds) between each slide. Or you can assign false as value to stop sliding

  8. data-duration="2000"

    This attribute specifies the duration (in milliseconds) of sliding.
  9. Note: If you are using more than one Sliders on one page or file, the first slider data-duration="" value will apply on all others sliders.
