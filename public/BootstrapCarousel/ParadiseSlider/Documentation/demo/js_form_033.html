mega vs single

You can use Paradise Slider CSS in two ways:

  • You can use only one Compressed or UnCompressed CSS file for all Paradise Slider layouts. You can find these files in the sub folder min in main css folder.
  • As well as concern to other method, you can use a single CSS file for a single Paradise Slider layout. You can find all of these files in the main CSS folder. In this case you can use only js_form_033.css file.

03 - Classes

We have used following our own classes and id in this layout, that control and change the whole slider.

#js_frm_033 : Describe the Slider's Unique Id.

.js_frm_signin : Enable Styles for black color sign in form

.js_frm_signin_white : Enable Styles for white color sign in form

.js_frm_signin_transparent : Enable Styles for transparent sign in form

.js_frm_signin_right : Move the form to the Right side

.js_frm_signin_center : Move the form in the Center

.slide : Enable Horizontal Sliding Effect.

.swipe_x: This class enables horizontal swipe function

.ps_easeOutQuint : This class enables Transition Timing Function for Slide Effect

.ps_control_smsquare : This class makes small and square, left and right control buttons

Note: js_frm Stands for " JavaScript Form " and 033 means " Version 033 ". And ps stands for " Paradise Slider ".

In this layout we have used left and right control buttons to move from one slide to another slide. You can move from one slide to another by clicking on these buttons or by dragging through the mouse button horizontally. We have created the following additional functions to control the slides:

  1. .slide

    This class enables effect of sliding.
    If you want to know about other animation effects of sliding please click HERE.

  2. .swipe_x

    This class enables swipe function. You can controll the slides by dragging the mouse button horizontally and in touch devices this class also enables the touch control.
    If you want to know about other functions click HERE.

  3. .ps_easeOutQuint

    This class enables Transition Timing Function for Slide Effect.
    If you want to know about other Transition Timing Function for Slide Effect please click HERE.

  4. .ps_control_smsquare

    This class makes small and square, left and right control buttons.
    If you want to know about other types of controls buttons please click HERE.

  5. data-pause="hover"

    This attribute pauses the slider from sliding through the next slide when the mouse pointer enters the slider, and when the mouse pointer leaves the slider it resumes the sliding.

  6. data-interval="4000"

    This attribute specifies the delay (in milliseconds) between each slide.

  7. data-duration="2000"

    This attribute specifies the duration (in milliseconds) of sliding.
  8. Note: If you are using more than one Sliders on one page or file, the first slider data-duration="" value will apply on all others sliders.
