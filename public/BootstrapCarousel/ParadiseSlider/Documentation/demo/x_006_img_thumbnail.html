mega vs single

You can use Paradise Slider CSS in two ways:

  • You can use only one Compressed or UnCompressed CSS file for all Paradise Slider layouts. You can find these files in the sub folder min in main css folder.
  • As well as concern to other method, you can use a single CSS file for a single Paradise Slider layout. You can find all of these files in the main CSS folder. In this case you can use only x_006_img_thumbnail.css file.

paradise slider

We have used following our own classes and id in this layout, that control and change the whole slider.

.swipe_y: This class enables Vertical swipe function

.thumb_scroll_y: This class enables Vertical scroll function for indicators when indicators cross the Width limit of slider. This class will use in only those layouts where indicators will be used.

.x_006_img_thumbnail_text : Enables setting of Left Text layer

.x_006_img_thumbnail_text_right : Enables setting of Right Text layer

.x_006_img_thumbnail_text_center : Enables setting of Center Text layer

.x_006_img_thumbnail_control_button : This class enables the setting of Left And Right Contorl Buttons

.x_006_img_thumbnail_indicators : This class enables the setting of Thumbnails Indicators

.ps_easeOutInCubic : This class enables Transition Timing Function for Slide Effect

In this layout we have used Two types of navigations first Images Thumbnails Indicators, second left and right control buttons to move from one slide to another slide. You can move from one slide to another by clicking on these thumbnails or by dragging through the mouse button Vertically. We have created the following additional functions to control the slides:

  1. .swipe_y

    This class enables swipe function. You can controll the slides by dragging the mouse button Vertically and in touch devices this class also enables the touch control.
    If you want to know about other functions click HERE.

  2. .thumb_scroll_y

    This class enables Vertical scroll function for indicators.
    If you want to know about other functions click HERE.

  3. .ps_easeOutInCubic

    This class enables Transition Timing Function for Slide Effect.
    If you want to know about other Transition Timing Function for Slide Effect please click HERE.

  4. .x_006_img_thumbnail_indicators

    This class makes Thumbnails Indicators of the slider.
    If you want to know about other types of controls buttons please click HERE.

  5. data-pause="hover"

    This attribute pauses the slider from sliding through the next slide when the mouse pointer enters the slider, and when the mouse pointer leaves the slider it resumes the sliding.

  6. data-interval="8000"

    This attribute specifies the delay (in milliseconds) between each slide.

  7. data-duration="2000"

    This attribute specifies the duration (in milliseconds) of sliding.
  8. Note: If you are using more than one Sliders on one page or file, the first slider data-duration="" value will apply on all others sliders.
