mega vs single

You can use Paradise Slider CSS in two ways:

02 - Slider HTML Elements

Bootstrap Carousel Contents

Bootstrap Carousel is the foundation of this slider. The main structure of Bootstrap Carousel which is used in this layout is as following:


    <div id="myCarousel" class="carousel slide" data-ride="carousel" data-pause="hover" data-interval="5000">

        <!-- Wrapper For Slides -->
        <div class="carousel-inner" role="listbox">

            <!-- Slide -->
            <div class="item">

                <img src="image.jpg" alt="image" />

            </div>

        </div

    </div>
Note: Remember if you are using more than one slider on same page the id of all sliders must be different from each others. In the above example myCarousel is id of slider.

paradise slider

We have used following our own classes and id in this layout, that control and change the whole slider.

.swipe_x: This class enables Horizontal swipe function

.thumb_scroll_x: This class enables Horizontal scroll function for indicators when indicators cross the width limit of slider. This class will use in only those layouts where indicators will be used.

.w_shop_103 : Enables General setting of slider

.w_shop_103_control_button : This class enables the setting of Navigation Buttons

.w_shop_103_indicators : This class enables the setting of indicators

.ps_easeOutSine : This class enables Transition Timing Function for Slide Effect

.w_shop_103_main_header: This class enables the setting of Header portion of the slider.

.w_shop_103_wrapper : Enables Wrapper Setting and Image setting

.w_shop_103_grid : This class apply setting on grid system of the bootstrap

.w_shop_103_icons: This class enables the setting of Icons portion of the Image Box.

.w_shop_103_image : This class enables Image Div setting of every box

.w_shop_103_footer : This class enables the setting of Item Name And Detail portion of Small Boxes.

.w_shop_103_big_footer : This class enables the setting of Item Name And Detail portion of Big Box.

.w_shop_103_rating : This class enables Setting of Price and Rating Stars for Small Boxes

.w_shop_103_big_rating : This class enables Setting of Price and Rating Stars for Big Box

.w_shop_103_discount :This class enables Setting of Discount Banner

.w_shop_103_new :This class enables Setting of New Banner

.w_shop_103_sale :This class enables Setting of Sale Banner

05 - Controls

Note: Please never use indicators in such layouts those move their boxes one by one or with the ratio of two By two. In these layouts only navigation left and right buttons will work fine. But you may use indicators and navigation left and right buttons both in only such layouts those slide their boxes all in all at the same time. 
    <!-- Indicators -->
    <ol class="carousel-indicators">
        <li data-target="#w_shop_103" data-slide-to="0" class="active">phones & tablets</li>
        <li data-target="#w_shop_103" data-slide-to="1">watches</li>
    </ol>

    <!-- Left Control -->
    <a class="left carousel-control" href="#w_shop_103" role="button" data-slide="prev">
      <span class="fa fa-caret-left"></span>
    </a>

    <!-- Right Control -->
    <a class="right carousel-control" href="#w_shop_103" role="button" data-slide="next">
      <span class="fa fa-caret-right"></span>
    </a>

In this layout we have used left and right control buttons to move the boxes. You can move boxes by clicking on these buttons or by dragging through the mouse button horizontally. We have created the following additional functions to control the slides:

  1. .swipe_x

    This class enables swipe function. You can controll the slides by dragging the mouse button Horizontally and in touch devices this class also enables the touch control.
    If you want to know about other functions click HERE.

  2. .thumb_scroll_x

    This class enables Horizontal scroll function for indicators when indicators cross the width limit of slider. This class will use in only those layouts where indicators will be used.
    If you want to know about other functions click HERE.

  3. .ps_easeOutSine

    This class enables Transition Timing Function for Slide Effect.
    If you want to know about other Transition Timing Function for Slide Effect please click HERE.

  4. .w_shop_103_control_button

    This class makes left and right navigation butttons.
    If you want to know about other types of controls buttons please click HERE.

  5. .w_shop_103_indicators

    This class makes indicators.
    If you want to know about other types of controls buttons please click HERE.

  6. data-pause="hover"

    This attribute pauses the slider from sliding through the next slide when the mouse pointer enters the slider, and when the mouse pointer leaves the slider it resumes the sliding.

  7. data-interval="8000"

    This attribute specifies the delay (in milliseconds) between each slide.

  8. data-duration="2000"

    This attribute specifies the duration (in milliseconds) of sliding.
  9. Note: If you are using more than one Sliders on one page or file, the first slider data-duration="" value will apply on all others sliders.
