mega vs single

You can use Paradise Slider CSS in two ways:

  • You can use only one Compressed or UnCompressed CSS file for all Paradise Slider layouts. You can find these files in the sub folder min in main css folder.
  • As well as concern to other method, you can use a single CSS file for a single Paradise Slider layout. You can find all of these files in the main CSS folder. In this case you can use only kenburns_065.css file.

02 - Slider HTML Elements

Bootstrap Carousel Contents

Bootstrap Carousel is the foundation of this slider. The main structure of Bootstrap Carousel which is used in this layout is as following:


    <div id="myCarousel" class="carousel slide" data-ride="carousel" data-pause="hover" data-interval="15000">

        <!-- Wrapper For Slides -->
        <div class="carousel-inner" role="listbox">

            <!-- Slide -->
            <div class="item">

                <img src="image.jpg" alt="image" />

            </div>

        </div

    </div>

We have assigned our own id to the first <div> that is kenburns_065 (KenBurns_065)

Slide Contents

The content for the slide is as following:

  • In bootstrap carousel <div> element which has item as class name show a single slide.
  • In this layout we have created our own <div> element which has kenburns_065_slide as class name for Text Layer 
      <!-- Slide -->
      <div class="item" style="background-image:url(paradise/images/kenburns_065_01.jpg);"> <!-- Image URL Here -->
        <!-- Left Slide Text Layer -->
        <div class="kenburns_065_slide">
          <h2>welcome to,</h2>
          <h1>codecanyon</h1>
          <a href="#">visit now</a>
        </div> <!-- /Left Slide Text Layer -->
      </div> <!-- /item -->
      <!-- End of Slide -->

In this layout we have used following components for Text Layer

  • <a>
  • <h1>
  • <h2>
  • <div>

03 - Classes

We have used following our own classes and id in this layout, that control and change the whole slider.

.swipe_x: This class enables Horizontal swipe function

#kenburns_065 : Describe the Slider's Unique Id.

.kenburns_065_slide : Enable Styles for Text Layer

.kenburns_065_slide_right : Move the Text Layer to the Right side

.kenburns_065_slide_center : Move the Text Layer in the Center

.thumb_scroll_x: This class enables Horizontal scroll function for indicators

.ps_indicators_gray : This class makes Gray indicators at the Top and the Right side of the slider

.ps_control_gray : This class makes left and right Gray butttons at the bottom and Left side of the slider

.slide : Enable Sliding Effect.

.ps_easeOutInCubic : This class enables Transition Timing Function for Slide Effect

.kbrns_rtl : This class enables Ken Burns Effect (Right To Left Moving)

Note: kenburns Stands for " Ken Burns " and 065 means " Version 065 ". And ps stands for " Paradise Slider ".

In this layout we have used two types of control buttons, left and right control buttons and indicators buttons to move from one slide to another slide. You can move from one slide to another by clicking on these buttons or by dragging through the mouse button horizontally. We have created the following additional functions to control the slides:

  1. .swipe_x

    This class enables swipe function. You can controll the slides by dragging the mouse button Horizontally and in touch devices this class also enables the touch control.
    If you want to know about other functions click HERE.

  2. .thumb_scroll_x

    This class enables Horizontal scroll function for indicators.
    If you want to know about other functions click HERE.

  3. .ps_easeOutInCubic

    This class enables Transition Timing Function for Slide Effect.
    If you want to know about other Transition Timing Function for Slide Effect please click HERE.

  4. .kbrns_rtl

    This class enables Ken Burns Effect (Right To Left Moving).
    If you want to know about other Ken Burns Effects please click HERE.

  5. .ps_indicators_gray

    This class makes Gray indicators at the Top and the Right side of the slider.
    If you want to know about other types of controls buttons please click HERE.

  6. .ps_control_gray

    This class makes left and right Gray butttons with icon at the Bottom and Left side of the slider.
    If you want to know about other types of controls buttons please click HERE.

  7. data-pause="hover"

    This attribute pauses the slider from sliding through the next slide when the mouse pointer enters the slider, and when the mouse pointer leaves the slider it resumes the sliding.

  8. data-interval="15000"

    This attribute specifies the delay (in milliseconds) between each slide.

  9. data-duration="2000"

    This attribute specifies the duration (in milliseconds) of sliding.
  10. Note: If you are using more than one Sliders on one page or file, the first slider data-duration="" value will apply on all others sliders.
