mega vs single

You can use Paradise Slider CSS in two ways:

  • You can use only one Compressed or UnCompressed CSS file for all Paradise Slider layouts. You can find these files in the sub folder min in main css folder.
  • As well as concern to other method, you can use a single CSS file for a single Paradise Slider layout. You can find all of these files in the main CSS folder. In this case you can use only video_096.css file.

02 - Slider HTML Elements

Bootstrap Carousel Contents

Bootstrap Carousel is the foundation of this slider. The main structure of Bootstrap Carousel which is used in this layout is as following:


    <div id="myCarousel" class="carousel slide" data-ride="carousel" data-pause="hover" data-interval="7000">

        <!-- Wrapper For Slides -->
        <div class="carousel-inner" role="listbox">

            <!-- Slide -->
            <div class="item">

                <img src="image.jpg" alt="image" />

            </div>

        </div

    </div>
Note: Remember if you are using more than one slider on same page the id of all sliders must be different from each others. In the above example myCarousel is id of slider. But we have assigned our own id to the first <div> that is video_096.

Slide Contents

The content for the slide is as following:

  • In bootstrap carousel <div> element which has item as class name show a single slide.
  • In this layout we have created our own <div> element which has video_096_img as class name for Background Images. HTML Video Tag does not support the mobiles, so these images will show on mobiles etc.
  • After that we have created our own <div> element which has video_096_vid as class name for Videos. This div contains the HTML5 video tag.
  • At the end we have created our own <div> element which has video_096_slide as class name for Text Layers.
      <!-- Slide -->

        <!-- Image For Mobiles. HTML Video Tag does not support the mobiles -->

In this layout we have used two types of navigations to move from one slide to another. We have created indicators dots and left, right navigation buttons. In this layout we have used following components for thumbnails:

  1. .swipe_x

    This class enables swipe function. You can controll the slides by dragging the mouse button Horizontally and in touch devices this class also enables the touch control.
    If you want to know about other functions click HERE.

  2. .thumb_scroll_x

    This class enables Horizontal scroll function for indicators when indicators cross the width limit of slider. This class will use in only those layouts where indicators will be used.
    If you want to know about other functions click HERE.

  3. .ps_easeOutInCubic

    This class enables Transition Timing Function for Slide Effect.
    If you want to know about other Transition Timing Function for Slide Effect please click HERE.

  4. .video_096_indicators

    This class makes indicators dots for the slider

  5. .video_096_control_button

    This class makes left and right navigation buttons for the slider

  6. data-pause="hover"

    This attribute pauses the slider from sliding through the next slide when the mouse pointer enters the slider, and when the mouse pointer leaves the slider it resumes the sliding.

  7. data-interval="7000"

    This attribute specifies the delay (in milliseconds) between each slide.

  8. data-duration="2000"

    This attribute specifies the duration (in milliseconds) of sliding.
  9. Note: If you are using more than one Sliders on one page or file, the first slider data-duration="" value will apply on all others sliders.
