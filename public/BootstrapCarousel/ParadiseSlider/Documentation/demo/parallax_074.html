mega vs single

You can use Paradise Slider CSS in two ways:

  • You can use only one Compressed or UnCompressed CSS file for all Paradise Slider layouts. You can find these files in the sub folder min in main css folder.
  • As well as concern to other method, you can use a single CSS file for a single Paradise Slider layout. You can find all of these files in the main CSS folder. In this case you can use only parallax_074.css file.

03 - Classes

We have used following our own classes and id in this layout, that control and change the whole slider.

.swipe_x: This class enables Horizontal swipe function

#parallax_074 : Describe the Slider's Unique Id.

.prlx_074_slide : Enable Styles for Text Layer

.prlx_074_slide_right : Move the Text Layer to the Right side

.prlx_074_slide_center : Move the Text Layer in the Center

.thumb_scroll_x: This class enables Horizontal scroll function for indicators

.ps_indicators_brown : This class makes Brown indicators box with numbers at the bottom and the center side of the slider

.ps_control_brown : This class makes left and right Brown Box butttons.

.slide : Enable Sliding Effect.

.ps_slowSpeedy : This class enables Transition Timing Function for Slide Effect

.prlx_074_bg_img : This class enables Setting of Background Image of Slider

.layer : This class enables Setting of Parallax Layers

Note: prlx Stands for " Parallax " and 074 means " Version 074 ". And ps stands for " Paradise Slider ".

In this layout we have used two types of control buttons, left and right control buttons and indicators buttons to move from one slide to another slide. You can move from one slide to another by clicking on these buttons or by dragging through the mouse button horizontally. We have created the following additional functions to control the slides:

  1. .swipe_x

    This class enables swipe function. You can controll the slides by dragging the mouse button Horizontally and in touch devices this class also enables the touch control.
    If you want to know about other functions click HERE.

  2. .thumb_scroll_x

    This class enables Horizontal scroll function for indicators.
    If you want to know about other functions click HERE.

  3. .ps_slowSpeedy

    This class enables Transition Timing Function for Slide Effect.
    If you want to know about other Transition Timing Function for Slide Effect please click HERE.

  4. .prlx_074_bg_img

    This class enables Setting of Background Image of Slider.

  5. .ps_indicators_brown

    This class makes Brown indicators box with numbers at the bottom and the Center side of the slider.
    If you want to know about other types of controls buttons please click HERE.

  6. .ps_control_brown

    This class makes left and right Brown Box butttons.
    If you want to know about other types of controls buttons please click HERE.

  7. data-pause="hover"

    This attribute pauses the slider from sliding through the next slide when the mouse pointer enters the slider, and when the mouse pointer leaves the slider it resumes the sliding.

  8. data-interval="5000"

    This attribute specifies the delay (in milliseconds) between each slide.

  9. data-duration="2000"

    This attribute specifies the duration (in milliseconds) of sliding.
  10. Note: If you are using more than one Sliders on one page or file, the first slider data-duration="" value will apply on all others sliders.
