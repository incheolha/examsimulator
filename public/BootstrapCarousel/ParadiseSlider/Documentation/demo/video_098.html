mega vs single

You can use Paradise Slider CSS in two ways:

  • You can use only one Compressed or UnCompressed CSS file for all Paradise Slider layouts. You can find these files in the sub folder min in main css folder.
  • As well as concern to other method, you can use a single CSS file for a single Paradise Slider layout. You can find all of these files in the main CSS folder. In this case you can use only video_098.css file.

02 - Slider HTML Elements

Bootstrap Carousel Contents

Bootstrap Carousel is the foundation of this slider. The main structure of Bootstrap Carousel which is used in this layout is as following:


    <div id="myCarousel" class="carousel slide" data-ride="carousel" data-pause="hover" data-interval="7000">

        <!-- Wrapper For Slides -->
        <div class="carousel-inner" role="listbox">

            <!-- Slide -->
            <div class="item">

                <img src="image.jpg" alt="image" />

            </div>

        </div

    </div>
Note: Remember if you are using more than one slider on same page the id of all sliders must be different from each others. In the above example myCarousel is id of slider. But we have assigned our own id to the first <div> that is video_098.

Slide Contents

The content for the slide is as following:

  • In bootstrap carousel <div> element which has item as class name show a single slide.
  • We have created our own <div> element which has video_098_vimeo as class name for Videos. This div contains the iframe of the video.
      <!-- Slide -->
      <div class="item">
        <div class="video_098_vimeo">
          <iframe src="https://player.vimeo.com/video/107469289?title=0&byline=0&portrait=0"
          webkitallowfullscreen mozallowfullscreen allowfullscreen></iframe>
        </div>
      </div> <!-- /item -->
      <!-- End of Slide -->

In this layout we have used following elements

  • <div>
  • <img>
  • <span>
  • <iframe>

Note: The active class needs to be added to the first slide. Otherwise, the carousel will not be visible.

In this layout we have used thumbnails of the videos and left and right navigation buttons to move from one slide to another. In this layout we have used following components for thumbnails:

  1. .swipe_x

    This class enables swipe function. You can controll the slides by dragging the mouse button Horizontally and in touch devices this class also enables the touch control.
    If you want to know about other functions click HERE.

  2. .thumb_scroll_x

    This class enables Horizontal scroll function for indicators when indicators cross the Width limit of slider. This class will use in only those layouts where indicators will be used.
    If you want to know about other functions click HERE.

  3. data-pause="hover"

    This attribute pauses the slider from sliding through the next slide when the mouse pointer enters the slider, and when the mouse pointer leaves the slider it resumes the sliding.

  4. data-interval="false"

    This attribute specifies the delay (in milliseconds) between each slide.

  5. data-duration="2000"

    This attribute specifies the duration (in milliseconds) of sliding.
  6. Note: If you are using more than one Sliders on one page or file, the first slider data-duration="" value will apply on all others sliders.
