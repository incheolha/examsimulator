how it works?

To trigger the modal window, you need to use a button, link, image, heading etc.

Then include the two data-* attributes:

  • data-toggle="modal" opens the modal window
  • data-target="#myModal" points to the id of the modal. In above example as you can see we have used data-target="#ps_modal_067". In this layout we have assigned #ps_modal_067 Id to the Modal.

The below examples are showing that how can we trigger modal by using the different methods besides Image Gallery


<a href="#" data-toggle="modal" data-target="#ps_modal_067">Open Modal</a> <!-- By Using Link -->
		
<img src="image.jpg" alt="" data-toggle="modal" data-target="#ps_modal_067"> <!-- By Using Image -->
		
<button type="button" data-toggle="modal" data-target="#ps_modal_067">Open Modal</button> <!-- By Using Button -->

03 - Classes

Modal

We have used following our own classes and id in this layout, that control and change the whole Modal.

#ps_modal_067 : Describe the Modal's Unique Id.

.modal: Describe the content as a modal and brings focus to it.

.fade : Enable Transition Effect which Fades the modal In and Out. Remove this class if you do not want this effect.

.ps_modal_zoomOut : Enable Zoom Out Effect for Modal. Without .fade class this Effect will not work.

.ps_modal_close_pink : This class makes Pink Color Close Button at the top and right side of the Modal.

.modal-dialog : This sets the proper width and margin of the modal.

.modal-content : The div which has .modal-content as a class name contains the Slider

Slider

We have used following our own classes and id in this layout, that control and change the whole slider.

.swipe_x: This class enables Horizontal swipe function

#thx_slider_modal_067 : Describe the Slider's Unique Id.

.thx_slider_modal_067_slide : Enable Styles for Text Layer

.thx_slider_modal_067_slide_right : Move the Text Layer to the Right side

.thx_slider_modal_067_slide_center : Move the Text Layer in the Center

.thumb_scroll_x: This class enables Horizontal scroll function for indicators

.thx_slider_modal_thumb : This class makes images thumbnails in the bottom of the slider

.slide : Enable Sliding Effect.

.ps_easeOutQuint : This class enables Transition Timing Function for Slide Effect

Note: thx Stands for " Thumbnail Horizontal (X) " and 067 means " Version 067 ". And ps stands for " Paradise Slider ".

    This class enables swipe function. You can controll the slides by dragging the mouse button Horizontally and in touch devices this class also enables the touch control.
    If you want to know about other functions click HERE.

  2. .thumb_scroll_x

    This class enables Horizontal scroll function for indicators.

  3. .ps_easeOutQuint

    This class enables Transition Timing Function for Slide Effect.
    If you want to know about other Transition Timing Function for Slide Effect please click HERE.

    This class makes images thumbnails in the bottom of the slider.
    If you want to know about other types of controls buttons please click HERE.

  6. data-pause="hover"

    This attribute pauses the slider from sliding through the next slide when the mouse pointer enters the slider, and when the mouse pointer leaves the slider it resumes the sliding.

  7. data-interval="4000"

    This attribute specifies the delay (in milliseconds) between each slide.

  8. data-duration="2000"

    This attribute specifies the duration (in milliseconds) of sliding.
  9. Note: If you are using more than one Sliders on one page or file, the first slider data-duration="" value will apply on all others sliders.
